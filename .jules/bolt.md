## 2024-05-24 - [Cache Machine Learning Models in Memory]\n**Learning:** In a Flask API handling video processing, reloading large machine learning models like OpenAI Whisper or RVM from disk into memory on every request creates a massive performance bottleneck and high memory pressure. \n**Action:** Use global dictionary caches (`_whisper_models = {}`, `_rvm_models = {}`) to ensure these models are loaded into memory only once per process, significantly improving request latency for repeated processing tasks.
