## 2024-05-03 - Path Traversal in Background Image API
**Vulnerability:** Found an unauthenticated, critical path traversal vulnerability in `web_app/api.py`. The `/api/jobs` endpoint concatenated unsanitized `input_filename` and `bg_image` input parameters from JSON requests directly into path configurations. `bg_image` even included logic to explicitly permit arbitrary absolute paths (`not os.path.isabs(bg_image)`), meaning an attacker could easily point to sensitive system files.
**Learning:** This occurred because validation was originally intended to be handled implicitly by the webform inputs in `/upload` endpoints, but direct JSON requests directly to `/jobs` bypassed that and allowed unsanitized inputs to reach core processing functions.
**Prevention:** Always enforce strict path validation using `secure_filename()` immediately when extracting variables from HTTP requests (both forms and JSON), and remove arbitrary absolute-path inclusions unless explicitly authenticated, constrained, and intended.
