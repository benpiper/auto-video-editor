# Project Overview

Auto Video Editor is an AI-powered automation suite for cleaning and enhancing video content.

## 🚀 Purpose
To eliminate the tedious manual work of editing out silences, filler words ("um", "uh"), and frozen frames from recordings, while providing advanced features like AI background removal.

## 🛠️ Tech Stack at a Glance
- **Core**: Python, OpenAI Whisper, FFmpeg, OpenCV, Mediapipe.
- **Web**: Flask, Vanilla JS, CSS Glassmorphism.

## 🏗️ Repository Structure
The project is organized as a multi-part monolith:
- **Core Processor**: The root directory contains the processing scripts and ML logic.
- **Web Portal**: The `web_app/` directory contains the Flask application and frontend.
